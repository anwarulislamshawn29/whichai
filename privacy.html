Privacy Policy

WhichAI ("we", "our") is committed to user privacy. Our extension is designed to be a local tool.

1. Data Collection

WhichAI does NOT collect personal data, browsing history, or keystrokes. All AI API interactions occur directly between the user's browser and the AI provider (OpenAI, Anthropic, or Google).

2. API Keys

User-provided API keys are stored locally using chrome.storage.local. These keys are never transmitted to our servers.

3. Third-Party Services

By using this extension, you are interacting with third-party LLM providers. Please refer to their respective privacy policies.
